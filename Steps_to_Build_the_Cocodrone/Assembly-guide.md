# Drone Specifications

| Component | Specification |
|---|---|
| Drone Type | Quadcopter |
| Frame | Custom 3D Printed |
| Flight Controller | Pixhawk / ESP32 |
| Motors | Brushless Motors |
| ESC | 30A ESC |
| Battery | 4S LiPo |
| Propellers | 1045 |
| Telemetry | WiFi / RF |
