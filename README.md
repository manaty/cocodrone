# CocoDrone

Open source coconut harvesting drone.

## About

CocoDrone is an open source project to build autonomous drones that safely and efficiently harvest coconuts — eliminating the dangerous practice of manual tree climbing.

**Website:** [cocodrone.net](https://cocodrone.net)

## Project Areas

- **Drone Platform** — Rugged multirotor frame for tropical environments
- **Harvesting Mechanism** — Lightweight end-effector to grip, cut, and lower coconuts
- **Computer Vision** — AI-powered ripe coconut detection and obstacle avoidance
- **Flight Controller** — Autonomous mission planning built on ArduPilot/PX4
- **Ground Station** — Offline-first mobile app for farmers
